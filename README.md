### How are graceful degradation and service workers related? (Answer this after doing the part below)
Graceful degradation is a way to ensure that when a portion of functionality is lost, there is still a way for the application to work properly, albeit in a limited manner. This prevents the application from crashing and losing all of its functionality. Service workers operate in a similar way in that they prevent a complete breakdown should one functionality be lost. Service workers use what is in the cache when the application does not have access to the internet. In both graceful degradation and service workers, they have a backup that prevents the application from losing functionality, instead operating from a limited scope.
